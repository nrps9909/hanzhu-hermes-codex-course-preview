需要會寫程式嗎?

不需要。課程先用 Hermes Desktop 應用程式完成看得見的成果，再慢慢補資料夾、規則、重跑與必要的進階設定。

上課要準備什麼?

帶可以登入與安裝工具的筆電，準備 Hermes Desktop App。模型帳號或 API key 依課前指引準備即可，LINE、Gmail 或 Google Drive 是後續練習場景，不是第一天的壓力來源。

可以用自己的工作案例嗎?

可以，但請先脫敏，不要帶密碼、客戶個資、合約全文、薪資、金流或公司機密。不確定時就改用課堂假資料。

開課資訊在哪裡?

日期、地點、費用、名額與報名連結先留空；確認後填入本頁的開課資訊欄。
